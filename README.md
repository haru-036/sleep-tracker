# zzz

Personal sleep tracking PWA. Record bed time, wake time, coffee intake, medication, and bath time.

## Features

- Simple sleep logging
- Coffee and medication tracking
- Sleep duration chart
- PWA support (works offline)

## Tech Stack

- React 19 + TypeScript
- Vite 7 + PWA
- Tailwind CSS v4
- Recharts

## Development

```bash
bun dev      # Start dev server
bun build    # Build
bun lint     # Lint
bun check    # Lint + Format
```
